# GDEVELOP ASTEROIDS

Aprenda a criar um dos jogos mais icônicos, o jogo Asteroids. Asteroids é um jogo de arcade com gráficos vetoriais muito popular lançado em 1979 pela Atari. O objetivo do jogo é destruir asteroides sem se deixar ser atingido por seus fragmentos.
